MySQL

Mysql指令

  1. SELECT：从数据库中提取数据

    • column_name(s):要查询的列

    • table_name:要查询的表

    • condition:查询条件（可选）

    • ORDER BY:排序方式，ASC表示升序，DESC表示降序（可选）

  2. INSERT INTO：用于向数据库表中插入新数据

    • table_name:要插入数据的表

    • column1,column2,...:要插入数据的列

    • value1,value2,...:对应列的值

  3. UPDATE：用于更新数据库表中的现有数据

    • table_name: 要更新数据的表
    • column1 = value1, column2 = value2, ...: 要更新的列及其新值
    • condition: 更新条件

  4. DELITE：用于从数据库表中删除数据

    • table_name: 要删除数据的表
    • condition: 删除条件

  5. CREATE TABLE：用于创建新的数据库表

    • table_name: 要创建的表名
    • column1, column2, ...: 表的列
    • data_type: 列的数据类型（如 INT、VARCHAR 等）
    • constraint: 列的约束（如 PRIMARY KEY、NOT NULL 等）

  6. ALTER TABLE：用于修改现有数据库表的结构。

    • table_name: 要修改的表
    • column_name: 要添加的列
    • data_type: 列的数据类型

    或：

    • column_name: 要删除的列

  7. DROP TABLE：用于删除数据库表

    • table_name: 要删除的表

  8. CREATE INDEX：用于创建索引，以加快查询速度

    • index_name: 索引的名称
    • column_name: 要索引的列

  9. DROP INDEX：用于删除索引

    • index_name: 要删除的索引名称
    • table_name: 索引所在的表

  10. WHERE：用于指定筛选条件

    • condition: 筛选条件

  11. ORDER BY：用于对结果集进行排序

    • column_name: 用于排序的列
    • ASC: 升序（默认）
    • DESC: 降序

  12. GROUP BY：用于将结果集按一列或多列进行分组

    • aggregate_function: 聚合函数（如 COUNT、SUM、AVG 等）

  13. HAVING：用于对分组后的结果集进行筛选

    • condition: 筛选条件

  14. JOIN：用于将两个或多个表的记录结合起来

    • JOIN: 可以是 INNER JOIN、LEFT JOIN、RIGHT JOIN 或 FULL JOIN

  15. DISTINCT：用于返回唯一不同的值

    • column_name(s): 要查询的列
